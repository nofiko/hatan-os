# مرجع ملفات الفلاش (نسخة محلية فقط)

هذه المجلدات **نسخة مرجعية** من الفلاشات — لا تُعدَّل الفلاشات الأصلية.

| المصدر | الوجهة هنا | الاستخدام |
|--------|------------|-----------|
| `G:\SteamOS\partsets` | `media-ref/steamos/partsets/` | كشف/تثبيت SteamOS |
| `G:\EFI\steamos\` | `media-ref/efi-steamos/` | إقلاع SteamOS |
| `D:\install-hatan.bat` | `media-ref/windows/` | تجهيز USB من Windows |

على الجهاز بعد التثبيت: `/opt/hatan-os/media-ref/`

ISO كامل لـ Windows (`D:\sources`) **لا يُنسخ** — ضعه على USB أو في:
`/var/lib/hatan/iso/windows.iso`
